# Python Mini Projects

This directory contains Python mini-projects created while learning Python fundamentals, functions, problem-solving, and program design.

## Projects

### Ticket Booking System

A console-based ticket booking application that simulates ticket reservation and management.

### ATM Simulator

A simple ATM simulation that performs basic banking operations such as balance checking, deposits, and withdrawals.

### Contact Book

A contact management application for storing, searching, updating, and deleting contact information.

### Email Validator

A utility that validates email addresses using Python string operations and validation logic.

### Library Management System

A mini-project that simulates book management, borrowing, and return operations.

### Number Guessing Game

A game where the user attempts to guess a randomly generated number.

### Rock Paper Scissors

A console-based implementation of the classic Rock-Paper-Scissors game.

### Simple Calculator

A calculator that performs basic arithmetic operations.

### Student Grade System

A program that calculates and manages student grades based on input marks.

### To-Do List Application

A task management application for adding, viewing, and removing tasks.

## Skills Practiced

* Functions
* Conditional Statements
* Loops
* Lists and Dictionaries
* String Manipulation
* Input Validation
* Problem Solving
* Modular Programming

## Purpose

These projects were created to strengthen Python fundamentals before progressing to:

* Object-Oriented Programming (OOP)
* File Handling
* Exception Handling
* SQL
* Data Analysis
* Data Science
