# Ticket Booking System

A beginner Python project for calculating movie ticket prices based on:
- Day type
- Customer category
- Show timing

## Versions

### V1
Basic implementation using multiple hardcoded match-case combinations.

### V2
Improved implementation with reusable pricing logic and reduced repetition.

## Concepts Used
- Conditional Statements
- Match-Case Statements
- Variables
- Arithmetic Operations
- Input Handling
- Basic Validation Logic
